# Лабораторная работа 5

Система мониторинга температуры с HTTP API и веб-интерфейсом.

## Запуск

### Linux/macOS
```bash
./build.sh
```

### Windows
```cmd
build.cmd
```

Браузер откроется автоматически на `http://localhost:5010`

## Компоненты

- **Симулятор** — генерирует данные температуры
- **Сервер** — сохраняет в БД, предоставляет HTTP API (порт 8080)
- **Веб-клиент** — интерфейс с графиками (порт 5010)

## API

- `GET /current` — текущая температура
- `GET /stats?type=history|hourly|daily&start=<timestamp>&end=<timestamp>` — статистика

## Требования

- CMake 3.10+
- C++11
- Python 3 + Flask
- SQLite3 (загружается автоматически)
